现在我们看到pip install -r requirements.txt为Python安装Flask和Redis库，并且该应用程序打印环境变量NAME以及调用socket.gethostname（）的输出。 最后，因为Redis没有运行（因为我们只安装了Python库，而不是Redis本身），所以我们应该期望在这里尝试使用它会失败并产生错误消息。
